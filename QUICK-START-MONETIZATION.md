# Quick Start: Monetize on Leanpub

## 5-Minute Setup

### Step 1: Create Account (2 minutes)
1. Go to: https://leanpub.com/author_getting_started
2. Sign up with email
3. Verify email

### Step 2: Create Book (1 minute)
1. Click "Create a New Book"
2. Enter details:
   - **Title:** The Rust Odyssey: A C++ Developer's Journey
   - **Subtitle:** Mental Models for C++ Developers Learning Rust
   - **Author:** Robin George Koshy
   - **URL slug:** the-rust-odyssey

### Step 3: Connect GitHub (2 minutes)
1. Go to: Book Settings → Writing Mode
2. Select "GitHub"
3. Click "Authorize Leanpub"
4. Select repository: `robingkn/the-rust-odyssey`
5. Branch: `main`
6. Manuscript folder: `book/`
7. Save settings

**That's it!** Leanpub will read `book/Book.txt` and `book/Sample.txt` automatically.

### Step 4: Configure Pricing (30 seconds)
1. Go to: Book Settings → Pricing
2. Set:
   - Minimum: $9.99
   - Suggested: $19.99
   - Reader sets price: ✅ Yes

### Step 5: Generate Preview (1 minute)
1. Click "Preview" button
2. Wait for generation
3. Download and review PDF/EPUB
4. Fix any issues if needed

### Step 6: Publish (30 seconds)
1. Click "Publish" button
2. Confirm
3. Book is live at: https://leanpub.com/the-rust-odyssey

---

## What Happens Next

### Automatic Sync
- Leanpub watches your GitHub repo
- When you push to `main`, Leanpub can auto-regenerate
- Or manually click "Publish New Version"

### Updates
1. Edit files in GitHub
2. Commit and push
3. Go to Leanpub dashboard
4. Click "Publish New Version"
5. Readers get notified of update

### Sales
- Leanpub handles payments
- You get paid monthly (via PayPal or direct deposit)
- Leanpub takes ~20% fee
- You keep ~80%
