# The Rust Odyssey v1.0.0

**Release Date:** January 17, 2026

## About

A technical book for experienced C++ developers learning Rust. This book translates C++ mental models into Rust mental models, focusing on where your intuition transfers cleanly and where it breaks down.

## Formats

- **PDF** - Full book in PDF format
- **EPUB** - E-reader compatible format
- **HTML** - Single-page HTML version

## What's Included

- 12 core chapters covering ownership, borrowing, lifetimes, traits, and more
- 3 appendices with practical reference material
- All code examples compile on Rust 2021 stable
